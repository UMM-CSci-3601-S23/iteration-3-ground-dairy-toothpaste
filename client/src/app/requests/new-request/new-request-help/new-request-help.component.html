How To Use Item Request Form

Item Category

The Item Category comes with 3 options you can choose from. Those being Foods, Toiletries or Other. Other is used when theres no category that fills the item you would like to request

Food Category

The Food Category comes with 5 options you can choose from. The food category will be available if the Item type you select is a Food. Depending on what food you want to request you will select the appropriate category.
